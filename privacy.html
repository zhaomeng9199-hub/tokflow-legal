Privacy Policy

This application collects limited TikTok user data required for video publishing and scheduling.

What we collect

  • Basic TikTok profile information
  • Video publishing permissions
  • OAuth access tokens (securely stored)

How we use data

Data is used only for scheduling and publishing videos to TikTok accounts.

Data sharing

We do not sell, share, or transfer user data to third parties.

Security

All tokens and credentials are stored securely and only used for authorized API requests.

User rights

Users can revoke TikTok access at any time via TikTok account settings.
