Terms of Service

This application is a video scheduling and publishing system integrated with TikTok APIs.

Users can upload, schedule, and publish short-form videos to TikTok accounts.

API Usage

This app uses TikTok Content Posting API to enable video publishing and scheduling functionality.

Authentication

Users authenticate through TikTok OAuth authorization flow. Access can be revoked at any time from TikTok settings.

Data Usage

We only access the minimal TikTok user data required for publishing videos and scheduling posts.

Data Storage

No personal data is sold or shared. Video metadata is used only for scheduling and publishing purposes.
