Terms of Service

This application is a video scheduling and publishing system integrated with TikTok APIs.

Users can upload, schedule, and publish short-form videos through authorized TikTok accounts.

Data Usage

This application only accesses TikTok user data required for content publishing, including basic profile information and video publishing permissions.

API Usage

We use TikTok Content Posting API to enable scheduled video publishing functionality.

User Authorization

All TikTok data access is performed through official OAuth authorization. Users can revoke access at any time from their TikTok account settings.

Data Storage

We do not sell or share user data. Video metadata is stored only for scheduling and publishing purposes.
